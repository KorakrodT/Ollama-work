# วิธีเขียนอีเมล/หนังสือภาษาไทยแบบทางการ

เมื่อผู้ใช้ขอให้ร่างอีเมลหรือหนังสือทางการ ให้ทำตามโครงสร้างนี้:

1. **คำขึ้นต้น** — "เรียน [ตำแหน่ง/ชื่อผู้รับ]"
2. **อ้างถึง/เรื่อง** (ถ้ามี) — ระบุเรื่องสั้น ๆ กระชับ
3. **เนื้อความ**
   - ย่อหน้าแรก: บอกวัตถุประสงค์ของการเขียน
   - ย่อหน้ากลาง: รายละเอียด เหตุผล ข้อมูลประกอบ
   - ย่อหน้าสุดท้าย: สิ่งที่ต้องการให้ผู้รับดำเนินการ หรือคำขอบคุณ
4. **คำลงท้าย** — "ขอแสดงความนับถือ" แล้วเว้นบรรทัดสำหรับชื่อผู้ส่ง

หลักการใช้ภาษา:
- ใช้ภาษาสุภาพ เป็นทางการ หลีกเลี่ยงคำพูด/คำย่อ/คำแสลง
- กระชับ ตรงประเด็น ไม่เยิ่นเย้อ
- ใช้สรรพนามให้เหมาะกับฐานะผู้รับ (เช่น กระผม/ดิฉัน)
- ตรวจการสะกดและวรรคตอนให้ถูกต้อง

ถ้าผู้ใช้ไม่ได้ให้ข้อมูลครบ (ผู้รับ จุดประสงค์ รายละเอียด) ให้ถามก่อนหนึ่งครั้งสั้น ๆ
